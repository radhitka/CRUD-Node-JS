# Rest API User

Simple CRUD using Node JS

## Setup

To deploy this project

copy `.env.example` to `.env`

from `db` folder run sql file to your database

run

```bash
npm install
```

and for run it

```bash
node index
```

or

```bash
nodemon index
```
